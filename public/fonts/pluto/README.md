# Pluto Font Files

Pluto is a licensed typeface by Neil Summerour / Positype.
Purchase at: https://www.myfonts.com/collections/pluto-font-positype

## Files expected in this directory

Drop the following files here after purchase. The @font-face declarations in
`src/index.css` are already wired to these exact filenames.

| File | Weight | Usage |
|---|---|---|
| `PlutoLight.woff2` + `.woff` | 300 | Light headings |
| `PlutoRegular.woff2` + `.woff` | 400 | Regular headings |
| `PlutoMedium.woff2` + `.woff` | 500 | heading-md, heading-sm |
| `PlutoBold.woff2` + `.woff` | 700 | display-md, heading-xl, heading-lg |
| `PlutoBlack.woff2` + `.woff` | 900 | display-xl, display-lg |

## Until files are added

Montserrat (loaded via Google Fonts) is the fallback and will render instead.
The UI is fully functional — only the brand typeface is missing.
